# 山河异闻计划 · 时代与历史

> 不要原创朝代。真实历史就是最大的世界观。
> 读者查得到的东西，我们不编。
> 史书没写的，都是我们的。

---

## 一、核心时代

故事发生在**明朝中后期**，主要跨度：

**嘉靖朝（1522—1566）→ 万历朝（1573—1620）→ 泰昌朝（1620）→ 天启朝（1621—1627）→ 崇祯朝（1628—1644）**

这120余年，是大明从表面繁华走向崩塌的全过程。
也是怪谈最多、民间最复杂、人心最动荡的时代。

---

## 二、为什么要选这个时代

| 理由 | 说明 |
|------|------|
| 真实感 | 嘉靖、万历都是读者熟悉的历史人物，写进去自动有真实感 |
| 缝隙够大 | 嘉靖修道、万历怠政、宦官专权——朝廷的混乱给民间留了巨大空间 |
| 民间活跃 | 这个时期商品经济发达，江湖百业兴旺，写什么都合理 |
| 怪谈温床 | 灾荒、战乱、瘟疫、流民——人心不稳，怪谈自然多 |
| 结局已知 | 崇祯亡国是所有人知道的结局，读者会带着"末日感"读每一个故事 |

## 三、历史大年表（框架）

### 嘉靖朝（1522—1566）
- **嘉靖元年（1522）**：嘉靖帝即位，大礼议之争。
- **嘉靖二十一年（1542）**：壬寅宫变，宫女谋杀嘉靖未遂。
- **嘉靖二十九年（1550）**：庚戌之变，俺答汗兵临北京城下。
- **嘉靖三十四年（1555）**：嘉靖大地震（华县地震），死亡人数据估八十余万。
- **嘉靖年间**：倭寇肆虐东南沿海。嘉靖沉迷修道，二十年不上朝。

**可利用的"缝隙"：**
- 大地震后流民四起，怪谈丛生。
- 倭寇之乱中沿海百姓的生死故事。
- 宫廷秘事——壬寅宫变的真相到底是什么？

### 万历朝（1573—1620）
- **万历元年至十年（1573—1582）**：张居正改革，中兴。这是大明最后的好时光。
- **万历十五年（1587）**：黄仁宇笔下的转折之年，表面平静，暗流涌动。
- **万历三大征**：宁夏之役（1592）、播州之役（1600）、朝鲜之役（1592—1598）。
- **万历中后期**：万历怠政三十年，朝堂空转。民间反而异常活跃。
- **万历四十三年（1615）**：梃击案。太子宫中遭刺客。
- **万历四十八年（1620）**：万历驾崩。

**可利用的"缝隙"：**
- 张居正死后被清算，他的旧部和门生何去何从？
- 朝鲜之役，大明将士远征，归来的人带了什么故事？
- 万历不上朝的三十年，地方上谁在管事？
- 三大案（梃击、红丸、移宫）的背后真相。

### 泰昌朝（1620）
- 泰昌帝朱常洛在位仅二十九天，暴毙。红丸案。
- 这是一个只有一个月的朝代，充满了阴谋和死亡的气味。

**可利用的"缝隙"：**
- 一个月的皇帝，宫里到底发生了什么？
- 二十九天够发生多少怪事？

### 天启朝（1621—1627）
- 天启帝朱由校，沉迷木工，不理朝政。
- 魏忠贤权倾天下，东林党遭清洗。
- 天启大爆炸（1626年，王恭厂大爆炸），至今原因不明。

**可利用的"缝隙"：**
- **天启大爆炸**——这本身就是最好的怪谈素材。一次神秘的大爆炸，数万人死亡，原因至今未明。这可以是整个系列的一个核心谜团。
- 魏忠贤的时代，民间是什么样子？
- 东林党人的命运。
- 辽东战事。努尔哈赤崛起。

### 崇祯朝（1628—1644）
- 崇祯帝勤政却无力回天。
- 大旱、蝗灾、瘟疫连年不断。
- 李自成、张献忠起义。
- 1644年，李自成入京，崇祯自缢于煤山。
- 大明亡。

**可利用的"缝隙"：**
- 大明最后的日子里，普通人是怎么过的？
- 李自成进京前夜，京城百姓在想什么？
- 崇祯上吊那棵树下，后来发生了什么？
- 南明的残余势力。
- 入关后的剃发令——这条线可以写无数悲剧。

## 四、我们的创作原则

1. **大事件不改。** 历史上发生的事，就是发生了。我们不创造架空历史。
2. **小事件可以加。** 史书没记录的，都是我们的创作空间。
3. **人物可以虚构，但不与历史矛盾。** 虚构人物不能改变历史走向。
4. **用真实细节增加沉浸感。** 年号、官职、地名、物价、节气——越真实越好。
5. **历史的缝隙，就是怪谈的温床。** 我们不写"穿越者改变历史"，我们写"历史缝隙里那些说不清的事"。

## 五、各时期的整体氛围

| 时期 | 氛围 | 关键词 |
|------|------|--------|
| 嘉靖 | 表面秩序还在，底下暗流涌动 | 修道、倭寇、地震、流民 |
| 万历前十年 | 最后的中兴，尚有一丝希望 | 张居正、改革、秩序 |
| 万历中后期 | 表面繁华，人心已散 | 怠政、商业、市井、纸醉金迷 |
| 泰昌 | 短暂而诡异 | 暴毙、阴谋、一个月 |
| 天启 | 黑暗、压抑、疯狂 | 魏忠贤、大爆炸、木匠皇帝 |
| 崇祯 | 末世感，所有人在劫难逃 | 天灾、流民、亡国、煤山 |

**每个时期，都适合不同类型的故事。**

嘉靖适合写"秩序表面下的裂缝"。
万历适合写"繁华中的孤独"。
天启适合写"黑暗中的挣扎"。
崇祯适合写"末日众生相"。

这就是我们的舞台。
