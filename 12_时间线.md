# 山河异闻计划 · 时间线

> 这个文件管理所有故事的时间坐标。
> 确保不同故事之间的事件不会矛盾。
>
> 历史坐标应同时核对 `17_世界年表.md` 与 `SOURCES.md`；年号与公历跨年时必须双重标注。

---

## 一、时间标注规则

每条时间线记录使用以下格式：

```
[年号][年份]年[月份] — [事件描述] — [来源故事编号]
```

例如：
```
嘉靖三十四年 五月 — 长江空船事件 — S002
天启六年 五月初六 — 王恭厂大爆炸 — S005
```
