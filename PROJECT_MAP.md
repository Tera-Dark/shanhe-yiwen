## 七、按任务读取

| 任务 | 必读 |
|---|---|
| 新建故事 | 宪法 → 00 → 22 选体例 → 20 → 09/08 → 11 → stories |
| 勾勒江湖 | 22 → 03/06/07/08 → `archive/stories/anthologies/江湖四件套_开工草案.md`（仅参考） |
| 写人物 | 宪法 → 05 → 相关职业/组织 → 故事人物档案 |
| 写历史事件 | 02 → 17 → SOURCES → 相关专设 |
| 写地方生活 | 08 → 15 → 18 → SOURCES |
| 写怪谈 | 01 → 04 → 09 → 19 → 20 |
| 写对白 | 20「对白声纹」→ `stories/桥头震后/P001_桥上第二碗/06_角色声纹表.md`（模板） |
| 审查故事 | 20 交稿检查 → 12 时间线 → 13 伏笔 → 19 未解之谜 → 21 |
| 修改文档 | AGENTS → docs/README → PROJECT_MAP → CHANGELOG |

路径前缀：设定一律 `docs/<类>/`；故事一律 `stories/<地脉组>/`。
