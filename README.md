# 山河异闻计划（Project ShanHe）

> 我们不是创作一本小说。而是创造一个能够承载无数故事的世界。

---

## 简介

**山河异闻**是一个以明朝中后期（嘉靖—崇祯，1522—1644）为背景的原创 IP 宇宙。

核心特色：真实历史为骨架，民间怪谈为血肉，普通人的命运为灵魂。

我们追求的不是"爽"，而是"余味"。

---

## 世界观核心

| 概念 | 说明 |
|------|------|
| 时代 | 明朝中后期（嘉靖→万历→泰昌→天启→崇祯） |
| 超自然体系 | "执念"——永远不解释是真是假 |
| 叙事风格 | 不解释、不说教、不站队、不定义善恶 |
| 最高原则 | 世界不会因为主角离开就停止运转 |
| 创作底线 | 所有怪谈都存在第二种解释 |

## 版本管理

本项目使用语义化版本号：

- **v0.x** — 世界观搭建阶段
- **v1.0** — 世界观基本完成
- **v1.x** — 故事创作阶段
- **v2.0** — 第一卷完成

每个文件使用独立的 Version 标注（头部 YAML frontmatter）。
重要变更记录在 `CHANGELOG.md` 中。

## 如何使用

### 写故事前

1. 阅读 `宪法.md`（最高准则）
2. 阅读 `00_创作理念.md`（创作纲领）
3. 查阅相关设定文件
4. 检查 `11_时间线.md` 确保时间不冲突
5. 检查 `102_伏笔与未解之谜.md` 确保不触碰核心谜团

### 和 AI 协作

将相关文件作为上下文上传给 AI，例如：

> "今天讨论江湖生态。"
> 上传：`03_江湖生态.md`、`01_世界底层规则.md`

AI 在开始工作前，应阅读 `宪法.md` 和 `00_创作理念.md`。

### 提交新设定

使用 Issue 模板提交新人物、新故事、新设定。
或直接提交 PR。
