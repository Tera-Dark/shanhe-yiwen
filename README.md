# 山河异闻计划 / Project ShanHe

> **名称状态：开发代号。** “山河异闻计划”“Project ShanHe”以及仓库名 `shanhe-yiwen` 均为项目管理代号，正式作品名尚未确定。

## 项目定位

这是一个以明朝中后期为主要历史背景的志异群像创作项目。

- **正篇主时间轴**：1522—1644，包含嘉靖、隆庆、万历、泰昌、天启、崇祯。
- **余波范围**：南明与清初可用于番外或人物结局，但必须标注为主时间轴之后。
- **历史原则**：大历史不改，虚构写在史料没有覆盖的人与事之间。
- **志异原则**：所有怪谈都保留至少一种理性解释；作者不确认超自然真相。
- **叙事原则**：世界大于主角，人物大于设定，情感大于反转。

## 核心约束

| 维度 | 规则 |
|---|---|
| 气质 | `00` 三句门楣；世界是容器，故事是主体 |
| 武学 / 超常 | 金庸式上限（`docs/世界/27`）；禁网文境界条；作者不确认妖鬼谱系；超常不替代人物选择 |
| 执念 | 是创作方法与民间解释，不是官方世界真相 |
| 历史人物 | 尊重其真实处境，不作为剧情工具人 |
| 怪谈 | 保留合理解释，作者不宣布最终答案 |
| 普通人 | 每个人都有立场、局限、关系和代价 |
| 语言 | 不直接使用未经考证的现代黑话、网络梗或后起民俗 |

## 史料等级

- **A**：同时代材料或可靠一手记录可证明。
- **B**：权威研究或较早的后世材料支持。
- **C**：近现代民俗记录、地方传说或流行说法。
- **D**：项目原创。
- **待考**：尚未找到可靠来源，不能直接进入正式正文。

项目允许虚构，但不允许把虚构伪装成史实。登记处：`docs/世界/SOURCES.md`。

## 写故事前

1. 读 `PROJECT_MAP.md`，再读 `docs/原则/宪法.md`、`docs/原则/00_创作理念.md`、`docs/原则/20_故事创作规范.md`。
2. 确定具体年份、地域和人物职业；选定**地脉组**与体例（见 22、23）。
3. 查询 `docs/世界/17_世界年表.md` 与 `docs/世界/SOURCES.md`。
4. 检查 `docs/管理/12_时间线.md` 和 `docs/素材/19_伏笔与未解之谜.md`。
5. 为历史、民俗和语言细节标注来源或【待考】。
6. 为异象准备超自然与理性两种解释。
7. 正文写完后运行
   `python3 scripts/postcheck_story.py stories/<地脉组>/<编号_标题>/正文.md`
8. 批量加肉后另跑 `python3 scripts/check_story_reuse.py`（跨章复写；补 R0 单文件盲区）。
9. 按 `docs/原则/21_故事审查标准.md` **v0.4** 做 **S0 + R0–R3**，并写本篇 `07_审查报告.md`（或合订总簿）。
10. 更新 `stories/catalog.json` 的 `review` 与组 README；主线可记 `docs/管理/13` 世界痕迹一行。
