## 引用约定

- 仓库内交叉引用写**自仓库根的完整相对路径**，例如 `` `docs/原则/宪法.md` ``。
- 网页卷宗经 `/project/docs/...` 读取；勿再写根目录下的 `00_xxx.md`。
- 新增设定：放入对应子目录，保留两位编号（若属 00–26 序列），并同步 `PROJECT_MAP.md`、`README.md`、`web/app.js` 卷宗列表。
