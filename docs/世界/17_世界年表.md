# 山河异闻计划 · 世界年表

> 不只是皇帝更替。
> 而是每一年发生过的大事、怪谈、灾荒、民谣、江湖传闻。
> 写单元时随时能查，时间线永远不会乱。

## 一、使用说明

这个年表比《11_时间线》更详细。
时间线管理的是"故事之间的时间关系"。
世界年表管理的是"世界本身发生了什么"。

每条记录格式：
```
[年号][年份]（公元） [月份（可选）]
类型：[历史/灾异/怪谈/江湖/民谣]
事件：[描述]
来源/备注：[出处或备注]
```

### 嘉靖元年（1522）
- **历史**：朱厚熜即位。大礼议之争开始。
- **江湖**：（待补充）

### 嘉靖三年（1524）
- **历史**：大礼议之争白热化。二百余名官员跪哭左顺门。

### 嘉靖七年（1528）
- **灾异**：黄河决口。

### 嘉靖十五年（1536）
- **历史背景**：东南沿海倭患持续，并在嘉靖中后期加剧。

### 嘉靖二十一年（1542）
- **历史**：壬寅宫变。宫女杨金英等十余人趁夜勒杀嘉靖未遂。
- **怪谈**：（宫变之后，宫里传言夜半有女子哭声。不知真假。）

### 嘉靖二十九年（1550）
- **历史**：庚戌之变。俺答汗率兵入关，兵临北京城下。

### 嘉靖三十四年十二月（公历1556年1月）
- **灾异**：**华县大地震**。传统记载常见死亡约八十三万的说法；具体数字与统计口径需在正式故事资料卡中注明来源。
- **怪谈**：长江漂来空船事件（002号怪谈）。
- **江湖**：地震后流民四起，瘟疫横行。各地怪谈激增。

### 嘉靖三十九年（1560）
- **灾异**：东南沿海倭患达到高峰。

### 嘉靖四十五年（1566）
- **历史**：嘉靖帝驾崩。裕王朱载垕即位，是为隆庆帝。

### 天启元年（1621）

## 八、使用原则

1. **写故事前先查年表。** 确保时间不冲突。
2. **发现了新事件就加进去。** 年表是活的。
3. **区分"史实"和"创作"。** 史实注明"历史"，创作注明"怪谈/江湖/民谣"。
4. **灾异是故事的温床。** 每一次灾荒背后都有无数故事。
