# 山河异闻 · 武学与奇术总纲

> **一句话：** 这是能写「一人当关、刀光血影」的武侠世界，不是修仙数值站。  
> 异闻（执念、说不清）与武学（可练、可传、可败）**两套账**，勿焊成一体。  
> 踏线人开卷仍是泥里走路的人；名武/绝武是世界的天花板，不是每章进度条。

配套：`docs/世界/01` · `04` · `docs/社会/03` · `07` · `docs/原则/25` · `AGENTS.md`

## 一、上限与气质（先钉死）

### 1.1 能写到哪儿

| 能 | 不能 |
|---|---|
| 一人破小队、破阵、敌一军前锋 | 移山填海、掐灭城池、改变大历史 |
| 碎金断铁、裂石、**破一座小山岗/石梁**（绝武极限，罕见） | 飞升、缩地成寸日常化、无限内力 |
| 纵屋脊、踏雪无痕、短距踏水（有代价） | 御剑飞行、瞬移、无敌金身永久 |
| 医毒、奇门小术、驭虫驭兽（有限） | 法术等级表、技能冷却、职业副本 |

**金庸感参照（创作语感，非抄书）：** 独孤、东方一类是**绝武点缀**；乔峰级是**名武可写**；镖师绿林是**行武常态**。全书绝武屈指可数，不可人人盖世。

### 1.2 四层武力（只给人贴标签，不给经验条）

| 层 | 俗称 | 能干什么 | 谁常见 |
|---|---|---|---|
| **凡武** | 防身、市井 | 棍棒刀口活命；打不过剪径也正常 | 沈陌开卷、脚夫、店家 |
| **行武** | 吃刀口饭 | 押镖、绿林、精捕；可连杀数人、护旗护货 | 冷旗、阿牛侧、老剪径 |
| **名武** | 一方有名 | 可一人破阵、敌百人而不溃；开山裂石偶见 | 少局主级、隐退老刀、外地来客 |
| **绝武** | 传说 | 破小山、碎精金、一人敌军——**事后才有人信** | 跨卷点缀；河东卷默认**不现本人** |

规则：

1. **层是旁人嘴上的账**，不是面板。人可「行武里偏强」，勿写「突破至名武 Lv.3」。  
2. **同层可互有胜负**——地、伤、药、人数、规矩都算数。  
3. **沈陌河东卷：** 目标仍在凡武→略懂行武规矩；**不以升级为章核**（见 25）。

### 1.3 与「异闻」分账

| | 武学 | 异闻 / 执念 |
|---|---|---|
| 来源 | 师承、苦练、药、机缘（人世） | 人心、误读、骗局、说不清 |
| 作者态度 | 可确认「他会这手」 | **不确认**唯一真相 |
| 进正文 | 刀、伤、汗、规矩 | 耳语、双解、勿夜行 |
| 禁止 | 把执念写成内功燃料 | 把碎金铁写成「妖力认证」 |

雨旗、齿印、土地缺脸——**优先走异闻/市井规矩**；勿用「内力显形」草率盖章。

## 二、武学五科（完整但不游戏化）

五科可兼修，**一门安身即可**。无「职业技能树解锁」。

### 2.1 外功（形）

- 拳脚、器械（刀枪剑棍、暗器）、身法架势。  
- 有**路数外号**（如「青驿旗下三步」「半截碑乱棍」），无段位证书。  
- 武戏写法：**脏、短、疼**——泥、血、喘；少金句招式念白。

### 2.2 内功（息）

- **是什么：** 调息、养伤、护体、延持、短时爆发。  
- **不是什么：** 打怪吸经验、境界突破动画、气海容量数字。  
- **获得：** 师承口诀 + 年月 + 药饵/静修；可走火、可废、可假。  
- **表现：** 刀砍在身上浅一分、夜路多撑一更、伤好得快——**旁人可以说是命硬**。

### 2.3 轻身（纵）

- 纵跃屋脊、窄梁、短距踏水/踏雪；长途仍靠脚与车。  
- 绝武可有「踏波半里」级传闻；日常写多了变卡通，慎用。

### 2.4 医毒（命）

- **金创、接骨、内伤、奇毒、解药**——独立饭碗（医馆、军医、绿林坐堂）。  
- 名医可与名武齐名；沈陌只会草药糊伤口级别。  
- 毒与暗器常绑绿林/刺客线；官差亦有验毒。

### 2.5 奇术（偏）

「妙法」放这里，**偏技巧、偏方、偏器**，不做法力表。

| 类 | 例（项目可用） | 边界 |
|---|---|---|
| 驭虫微术 | 萤火绕身照明、训蜂驱疫（小） | 非万虫潮灭城 |
| 声光障眼 | 火折、药粉烟、铜镜 | 非召唤天火 |
| 机关暗器 | 袖弩、绳套、车轴陷阱 | 要零件与准备 |
| 奇门小术 | 观星辨向、风水口彩 | 不保证灵验；可双解 |
| 幻迷 | 迷香、影戏、话术 | 破了就是破了 |

**萤火绕身**这类：写清是「养虫+香料+手法」，读者可当绝活，也可当戏法——**作者可确认会这手，不确认通神**。

## 三、门派与山头（饭碗，不是技能商店）

### 3.1 定义

**门派 / 局 / 帮 / 香火 / 绿林山头** = 师承 + 规矩 + 忌讳 + 银子来路 + 仇与盟。

先问四句：

1. 吃什么？（镖银、香油、截货、官饷……）  
2. 怕什么规矩？（旗不湿、出事莫告官……）  
3. 名怎么传？（路号、帖、海捕）  
4. 谁是钉子人物？（一人一魂，不写「门派战力总值」）

### 3.2 分层（与武力层交叉，不一一对应）

| 型 | 例（项目） | 武力重心 |
|---|---|---|
| 护送行 | 青驿镖局 | 行武为主，少名武坐镇 |
| 牙与商 | 康家、牙行 | 银与契；武力外雇 |
| 绿林 | 河东剪径窝 | 凡武～行武；狠在人多与路熟 |
| 官差半退 | 老成一类 | 制度残余 + 个人手段 |
| 隐世师门 | 跨卷再立 | 名武/绝武种子，勿河东开篇堆 |

**禁止：** 武林盟主选举、 convene 天下比武逼沈陌上场、门派贡献度。

### 3.3 河东卷默认可见

- **青驿镖局**（旗、锣、帖、雨旗规矩）  
- **康家酱园**（货主刀比雨快）  
- **牙行 / 柳先生**（契口）  
- **剪径与靴钉窝**（泥里见血）  
- **半截碑坑钱**（半退差）  
- 外地名武：只闻名、可擦肩，**第一幕不收徒沈陌**

## 四、战场与书写约定

1. **一人敌军：** 写「破前锋、乱阵脚、杀出条路」，少写无伤杀万人。  
2. **破小山 / 碎金铁：** 绝武或名武极限；事后众说纷纭（岩本有裂、铁本是脆……可留双解）。  
3. **沈陌视角：** 看不懂招式名也行；只懂疼、怕、逃、侥幸。  
4. **声纹：** 日常仍大白话；名武可以话少，勿玄学念经。  
5. **史料：** 武学体系标 **项目原创 D**；真军制官职另走 SOURCES。

## 五、与旧红线的关系（2026-07-21 校准）

| 旧表述（易误解） | 现行 |
|---|---|
| 「不建立修炼、法力、等级」 | **禁止**网文境界数字与可验证妖谱；**允许**武学四层+五科 |
| 「没有人拥有超能力」 | **禁止**法术超能；**允许**金庸式武功神话上限 |
| 「怪谈双解」 | **仍强制**于异闻；武学胜负可确认 |
| 踏线人成长 | **仍以**懂/债/名为主；武力偶进，不作章核 |

冲突时：宪法情感与人物 > 本总纲；本总纲 > 单篇炫技冲动。
