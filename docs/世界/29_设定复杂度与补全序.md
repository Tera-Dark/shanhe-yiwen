# 设定复杂度与补全序

> **目标：** 世界像真的有厚度，读者却不用背设定。  
> **手段：** 层少、钉硬、卡短、用则写、不用则种子。

配套：`27` 武学 · `28` 诸地 · `01`/`04` 异闻 · `stories/世界/`

---

## 一、精妙度三条

1. **一场戏最多压两本账**  
   四本账：`武` · `银` · `规矩` · `异闻`。  
   例：剪径 = 武+银；旗过桥 = 规矩+武；勿夜行耳语 = 规矩+异闻。  
   **禁止**一章同时秀绝武 + 全套医毒 + 厂卫制度 + 妖谱。

2. **卡有上限**  
   | 形态 | 行数建议 | 用途 |
   |---|---|---|
   | 种子 | ≤20 | 名+一句功能 |
   | 框架 | 20–40 | 吃饭/怕/一条未了结 |
   | 成档 | 40–80 | 可写正文对照；**超过 100 行拆长文进 docs** |
   | docs 总纲 | 一题一文件 | 27/28 类；禁止再开「百科全书」 |

3. **未开卷地脉只立气质**  
   运河/晋地/京畿：28 里三行够；**不**预写完整门派表、货币表、武学谱。开卷再立 README + 3 枚钉子。

## 三、高杠杆缺口（只补这些）

| 优先级 | 缺口 | 为何杠杆高 | 补法 |
|---|---|---|---|
| P0 | **银与脚程** | 每章都要花/付/欠 | 世界卡；粗量级即可 |
| P0 | **信息价** | 柳/茶白/老成全靠半句 | 世界卡；与牙契咬 |
| P0 | 河东核心钉加厚 | 卡仅 1 句，正文已超前 | 康家·牙·碑·旗·勿夜行 → 40 行内 |
| P1 | 医毒在河东的可见度 | 27 有科、正文只有热水 | 武力尺补半节；专卡可后开 |
| P1 | 厂卫/东厂 | 只占位；沈陌怕文书 | **保持种子**，京畿卷再开 |
| P2 | 全国货币·科举·军制 | 易变史实坑 | 不写；用 SOURCES 个案 |

**明确不做（复杂度陷阱）：**

- 完整「九大门派」「内功心法一百零八式」  
- 全国统一物价表、精确里数表  
- 妖鬼分类进化树  
- 为未写卷预造 BOSS 战力榜

## 四、四本账速查（写戏时）

| 账 | 问一句 | 河东例 |
|---|---|---|
| 武 | 谁能打过谁、要不要打 | 剪径 vs 锣；旗手不拔刀也压人 |

| 异闻 | 有没有第二种解释 | 齿印、缺脸土地、雨旗邪门 |
