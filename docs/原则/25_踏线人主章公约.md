# 踏线人主章公约

> **学燕云的「少东家」：跟脚、解锁地、主支分层。**  
> **不学：技能树、战力榜、世界围着一人转。**  
> 宪法仍在：世界大于主角。踏线人是**引路人**，不是开天辟地的神。

---

## 一、一句话

**踏线人带读者进地图；群像把时代踩实；地脉成卷，卷内分主线 / 副线 / 其他。**

| 角色 | 职责 |
|---|---|
| **踏线人**（全书主章 POV） | 走路、认规矩、欠人情、长见识；串卷 |
| **地脉钉子**（副线主角） | 冷旗、宋婆、阿段……一地之魂，可独立成篇 |
| **路人与行当** | 茶馆、牙行、捕快、酱园——厚度，不是工具人 |

## 二、踏线人是谁（现行定锚）

| 项 | 内容 |
|---|---|
| 名 | **沈陌** |
| 年岁气质 | 约弱冠前后；像走过几天真泥路的人，不像说书里的侠 |
| 来处 | 小地方出来；身上有一封须送的信（或一桩须了的人情）——**具体事由可卷内慢慢露，勿开场交代干净** |
| 武 | 开卷凡武（防身）；可知行武规矩；**不以名武/绝武为河东章核**；无境界数字（见 27 武学总纲） |
| 江湖称呼 | 初无名；河东之后或被人叫「走路的」「沈半程」——**名号是别人给的，不是自称升级** |
| 与冷旗 | 河东卷可同路、可擦肩；冷旗是钉子，沈陌是眼睛——**不夺半截杆的主** |

人设细档见 `docs/社会/05` 增补条或 `stories/河东线/R_沈陌/`（若立折）。

## 三、成长（只许这几样）

| 可成长 | 文本表现 | 禁止 |
|---|---|---|
| 识路 | 会认坑钱、旗净、夜不行、哪段泥咬人 | 小地图、任务栏、收集度 |
| 识人 | 知冷旗厚、柳滑、阿段肯说不 | 好感度条 |
| 识规矩 | 从踩雷到会递帖、会装糊涂 | 门派贡献度、境界突破动画 |
| 识武（轻） | 知道旗下三步、假脊、刀快在心里 | 一章拜师顿悟、战力榜 |
| 代价 | 欠人情、留疤、名号进某些人嘴里 | 越战越勇无成本 |
| 关系网 | 跨卷「另见」有人记得他 | 全图主角光环 |
| 轻武 | 多一招活命，少一次挨打 | 心法、内力、战力对比 |

**检验句**：读完一章，沈陌若「更懂」了世界，而不是「更强」了——过。若读者开始比谁刀快——停，改。

## 四、卷内三槽（发布与 catalog）

每卷（地脉）目录固定三槽：

```text
某卷
├─ 主线 track: main   ← 沈陌在本卷的脚程（体例多为 Z 主纪章，或标主线的 P/Q）
├─ 副线 track: side   ← 钉子完整故事（冷旗 P002、桥头 P001…）
└─ 其他 track: other  ← 奇遇/探游/传闻/残章/人物折/规矩帖
```

| 字段 | 说明 |
|---|---|
| `track` | `main` \| `side` \| `other` |
| `main_path` | 番茄**主连载**章序（沈陌主线 + 必要副线高峰） |
| `reading_path` / `roam_path` | **漫游序**（现有地脉跑图感，可与 main 不同） |

规则：

1. 副线必须**可独立成篇**；主线只路过或轻咬，不吞掉钉子的主。  
2. 其他槽丰富世界观；短则标幕间/残页（见 24）。  
3. 跨卷互动用人情债、半句名号、一封信——允许多年后再见，不绑架连续周更。

## 六、主线章写法（R-读加码）

- 单章舒适区：**约 3000–5000 汉字**（主线正章）；短于 2000 须标幕间或捆发。  
- 每章至少有一：**新地点钉 / 新规矩 / 新人一句站得住 / 沈陌一点「更懂」**。  
- 章末钩子：下一脚在哪，或人情未了，或名号未落。  
- 怪谈：双读可活，作者不站队（同 21/宪法）。  
- **禁止**：修炼突破、系统面板、无敌打脸流水、把群像写成经验包。

## 七、河东卷主线功能（样章锚）

沈陌第一张真地图 = 河东线。

建议功能节拍（可多章拆细）：

1. 上路：信/人情在身，泥咬鞋  
2. 半截碑：付钱，识老成，见「勿夜行」  
3. 雨旗过身：识青驿与冷旗气质  
4. 同路或旁观过桥：桥心钱、旗干——**不替冷旗做选择**  
5. 东市：牙契/酱园/茶馆三说轻扫  
6. 离线：人走，线还在；带走一点懂与一点债  

副线高峰仍归冷旗《半截杆》等；主线可「听说北岔」「见他无旗出门」，不改卷宗。

## 八、数百章怎么长出来（结构，不是一次写完）

| 层 | 规模感 | 内容 |
|---|---|---|
| 全书主线 | 可规划 **200–400 章槽** | 沈陌走多地脉；每卷 15–40 主线章 |
| 每卷副线 | 每卷 10–30 章 | 钉子弧，可拆章连载 |
| 其他 | 海量 | 奇遇/传闻/残页/探游，填厚度 |

机器源：`stories/主章线/目录骨架.md` + `catalog.json` 的 `main_path` / `track`。  
**骨架可先满，正文按卷填**——未写章在目录标「槽位」，勿假称已成稿。
