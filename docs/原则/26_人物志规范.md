# 人物志规范

> **R 人物折**是文学速写：一人一折，可当正文读。  
> **人物志**是档案体：设定 + 部分经历 + 关系网，给作者与读者查阅，**不抢主线戏**。  
> 配套：`22` 体例 · `23` 分组 · `25` 踏线人 · `05` 类型模板。

---

## 一、一句话

**折写魂，志立档；档可厚，戏不夺。**

| | 人物折 · R | 人物志 · N |
|---|---|---|
| 气质 | 说书人一折 | 案前档案 |
| 篇幅 | 0.1–0.5 万 | 0.3–1.5 万（可分卷增补） |
| 读者感 | 像故事 | 像可翻的卷宗 |
| 必写 | 饭碗 · 怕什么 · 未了结 | 身份表 · 经历摘 · 关系 · 禁写 |
| 编号 | `R###` / `R-SM` | `N-xxx`（人名缩写或流水） |
| catalog | `genre: R` | `genre: N` · `track: other` |
| 路径 | `stories/<地脉>/<R…>/` | `stories/人物志/<N_名>/` |

同一人可**既有折又有志**：折可入漫游/连载；志挂卷宗感浏览。志里细节不得与已成稿正文冲突；冲突以正文为准，改志不改正文核。

## 二、体例代号 · N

| 项 | 说明 |
|---|---|
| 代号 | **N**（档案 / 人物志） |
| 像什么 | 案牍、行状、半本记事——不是技能表、不是好感度 |
| 必有吗 | 核心钉子与踏线人优先；路人可只折不志 |
| 超常 | 不写可验证妖谱；「别人怎么说」可记，作者不盖章 |

---

## 三、档案必填结构（正文模板）

每份人物志 `正文.md` 按下列章节写（可增不可乱序砍必填）：

1. **名号表** — 本名 / 路名 / 乳名 / 文书名；谁叫、谁不叫  
2. **吃饭** — 行当、靠山、银从哪来、不吃哪类活  
3. **身量与气** — 外表一两笔；气质不写战力  
4. **怕什么** — 至少两条，一条落地到具体物/地  
5. **一件未了结** — 河东卷内会咬或只闻其声  
6. **经历摘** — 3–8 条时间锚（年号可虚；用「入河东前 / 河东卷内」分层）  
7. **关系网** — 表：人 · 关系 · 沈陌侧可见度  
8. **声口** — 口头禅、句长、回避方式（对齐 `20`）  
9. **河东卷内功能** — 主线/副线/铺地；**禁止抢谁的戏**  
10. **禁写清单** — 本人物不得出现的写法（升级、圆满、唯一真凶等）  
11. **另见** — 已成稿条目 ID  

frontmatter 必含：`genre: N` · `people` · `group`（主属地脉）· `volume` · `track: other` · `links`。

## 四、与主线、副线的边界

| 规则 | 说明 |
|---|---|
| 世界 > 主角 | 志写清此人自己的命，不写「为沈陌服务」 |
| 钉子不夺 | 冷旗半截杆、阿段刀鞘、宋婆第二碗——主在副线/旁纪 |
| 经历摘可剧透档 | 志可写读者尚未读到的「将发生功能」，但标【规划】；成稿后改【已现】 |
| 不写满 | 未了结保持裂缝；禁止志里结案 |
| 史料 | 项目原创 D；真年号/制度须 SOURCES 或【待考】 |

## 六、中篇河东与人物志的用法

河东卷目标体量 **约 30–60 万汉字量级**（主线 Z 厚写 + 副线/其他铺地；槽先满，按幕填）。

人物志在写作中的用法：

- 开写新 Z 前，扫相关 N 的「禁写」与「未了结」  
- 新经历进正文后，回填志「经历摘」一条  
- 番茄 main_path **默认不塞 N**；N 进 `roam_path` 或卷页「图鉴/人物」槽（前端可后接）
