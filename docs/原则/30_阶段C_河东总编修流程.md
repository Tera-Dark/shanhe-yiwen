# 阶段 C · 河东卷真实总编修流程

> **前置：** 阶段 A（审印真相）· 阶段 B（去伪肉 / path 交错）· v0.44 治理收口。  
> **本阶段目标：** 不是让每章更长，而是让每章发生**别章不能替代**的事。  
> **配套：** `21` 审查 · `25` 踏线人 · `20` 创作规范 · `stories/河东线/阶段C_手术矩阵.md`

---

## 〇、一句话

**治理完成 ≠ 文学完成。先矩阵，再样板，主笔确认后才推广。禁止一次性批量改 HD00—Z040。**

---

## 一、有效新增门槛（硬）

一段新文字进入正文，须**至少满足下列两项**：

| 项 | 含义 |
|---|---|
| 新事件 | 现实中发生了可叙述的事 |
| 新选择 | 沈陌（或关键人物）做了与章首不同的决定 |
| 新代价 | 当场或后文失去银/路/名/人/体面/活 |
| 新关系 | 谁更信 / 更恨 / 开始利用 / 不再用他 |
| 新信息 | 后文会用到的事实（须在矩阵写回声章） |
| 生活细节 | 且**改变行为**（非纯风景） |

仅有气氛、感慨、同义反复、项目金句 → **不算**有效新增，删除或并入动作。

### 搬章检验（必做）

1. 这段能不能**原样搬到另一章**？能 → **删**。  
2. 它改变了谁、损失了什么、后文还会不会响？全无 → **删**。

---

## 二、工作顺序（门闸）

```text
① 确认 v0.44+ 治理未回滚（reuse PASS · status≠seal）
② 轻量手术矩阵（每章 ≤ 一行功能表，禁止五十列表格表演）
③ 仅四章样板精修（默认 Z001 / Z013 / Z019 / Z038）
④ 校验 + 样板对比报告
⑤ 停。主笔通读后，才允许「分幕推广」下一批
```

| 门闸 | 未过则 |
|---|---|
| 矩阵未提交 | **不得**改正文 |
| 样板四章未停 | **不得**动其余主线章 |
| 主笔未点头推广 | **不得**批量改整卷 |
| reuse / postcheck 非 0 | **不得**交稿 |

可选第四样板替换（主笔偏好）：`Z014`（人情外环）或 `Z030`（离线）——替换须在矩阵与报告中写明，仍只改四章。

## 三、绝对禁止

1. 脚本生成正文段落；跨章复制或同义改写模板。  
2. 统一补丁标题（泥里再一笔 / 檐下半盏 / ××之后的骨…）。  
3. 统一投放路边老人、檐下陌生人、茶客讲规矩。  
4. 每章追加懂/债/名清单或「不是 A，是 B」通用升华。  
5. 选择后二次同义解释；正文工程词（本幕/跨卷/主线/钉子/S0/章核…）。  
6. 自动 pending→朱注/验讫；重开桥头正章；恢复 P001 进 catalog。  
7. 擅自改总纲「30—60 万」数字（只可出 A/B 可行性说明）。  
8. 以字数达标作为完成标准。

## 四、轻量矩阵字段（每章一行够用）

见 `stories/河东线/阶段C_手术矩阵.md`。必填：

- 外部事件 · 选择 · 即时代价 · 延迟代价落点 · 手术建议（保留/精修/结构重写/合并/幕间）

**结构重写或合并候选**触发条件：

- 答不出「本章改变了什么」；  
- 仅有：听传闻 → 决定不查 → 想通一点 → 带着泥走。

## 六、文字层（辅助，非自动删词）

「不是 / 像 / 停 / 泥 / 债…」**不是禁词**。人工盯：

1. 一章最多 **一个** 门楣式判断句。  
2. 有力动作后不补「这意味着…」。  
3. 结尾优先：动作 / 物件 / 未答的话 / 现实损失 / 后续任务。  
4. 慎用：泥还在转、不是侠、停也是走法、欠着才能走（前文动作已表达则删字）。
