# 山河异闻计划 · 诸地风土志

> 地域不是今天的城市名片，也不是现代美食清单。
> 写明代地方，先写水土、交通、物产、行业、人口流动和当地人真正担心的事。

## 一、使用规则

1. 现代形成的名菜、曲艺、帮会和身份称谓，不得直接倒置到明代。
2. 食物先写主要粮食与物产；具体菜名需另附来源。
3. 地域气质只是入口，不能把所有当地人写成同一种性格。
4. 正式故事必须补充具体年份、行政区划和资料来源。

## 二、核心区域

### 北京（顺天府）
- **环境**：皇城、坊巷、城门、驿路和北方商旅。
- **物产方向**：麦面、杂粮、羊肉、酱菜等；具体菜名另考。
- **人群**：官员、军户、宦官、匠役、商人、脚夫、流民。
- **气质**：权力中心的沉默、宫墙外的传言、灾异后的集体恐惧。
- **提示**：八旗只可在1644年前后及余波中出现。

### 南京（应天府）
- **环境**：旧都、城墙、秦淮水系与长江交通。
- **物产方向**：稻米、河鲜、鸭馔、时蔬。
- **人群**：留都官署人员、军户、文人、商人、乐户、手工业者。
- **气质**：旧都记忆、繁华背后的失落。

### 苏州
- **环境**：水网、桥梁、园宅、织造与周边市镇。
- **物产方向**：稻米、鱼虾、菱藕、茶与细点。
- **人群**：织工、染工、商人、文人、戏班、船户。
- **气质**：精细生活中的裂缝。
- **提示**：可写昆曲及相关声腔；“评弹”作为成熟称谓需谨慎。

### 杭州
- **环境**：西湖、运河、寺院、山路与商业。
- **物产方向**：稻米、湖鲜、竹笋、茶叶。
- **人群**：僧人、茶户、船户、商人、游客、文人。
- **气质**：过分美丽的景物与无法放下的旧事。

### 扬州
- **环境**：运河、漕运、盐业与南北货物流动。
- **物产方向**：稻米、河鲜、腌腊和沿运河输入的货物。
- **人群**：盐商、官吏、船工、脚夫、乐户、贫民。
- **气质**：财富、人情债与繁华的不安全感。

### 山西
- **环境**：山地、黄土、驿路、边地贸易和矿冶区域。
- **物产方向**：粟、黍、麦食、羊肉、醋。
- **人群**：农户、商旅、矿工、军户、驿卒。
- **气质**：土地、宗族、远行与埋在地下的旧事。

### 四川
- **环境**：盆地、江河、井盐、山路与地方市场。
- **物产方向**：稻米、豆类、猪肉、井盐、花椒；辣椒传播须按年代考证。
- **人群**：农户、盐工、船户、商人、军户。
- **气质**：山路阻隔、盐井深处和难以传出的消息。
- **提示**：担担面、麻婆豆腐、近现代火锅和袍哥不作为明代通行背景。

### 江西
- **环境**：山地、河流、书院、茶区与景德镇瓷业。
- **物产方向**：稻米、茶、鱼、山货。
- **人群**：农户、窑工、商人、书生、船户。
- **气质**：窑火、书院、宗族与读书人的执念。

### 福建
- **环境**：山海相逼，沿海港口、内陆山路和海上贸易并存。
- **物产方向**：稻米、海产、茶、山货。
- **人群**：渔民、海商、船工、军户、沿海迁徙者。
- **气质**：海禁与出海、归乡与失踪。
- **提示**：不使用“华侨”等近现代身份概念；沙茶面、佛跳墙等后起菜名不进入明代正文。

## 三、故事资料卡必填项

- 故事年份与当时行政区划；
- 主要交通路线；
- 本地物价与计量；
- 方言或称谓来源；
- 灾害、战争和赋役背景；
- 至少一条可追溯资料来源。
