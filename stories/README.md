## 新条目怎么开

1. 选定 **地脉组**（或新建组 README）。
2. 选定 **体例编号**（Y/Q/R…）。
3. 建 `stories/<组>/<编号_标题>/`，复制 `_templates/条目_正文头.md` → `正文.md`。
4. 长旁纪可加 01–07 设定包。
5. 写完：`python3 scripts/postcheck_story.py <正文路径>` → 更新组 README + `catalog.json`。

## 旁纪完整包（可选）

```text
P00X_标题/
├── README.md
├── 01_设定.md … 07_审查报告.md
├── versions/
└── 正文.md
```

短体裁（Y/Q/W/R/G/C）通常 **仅需 README + 正文**。
