# 故事目录

本目录存放已经进入实际创作流程的单元故事。

## 结构规范

每篇故事使用独立编号目录：

```text
SXXX_工作标题/
├── README.md          # 故事卡、状态与文件导航
├── 01_设定.md         # 情绪目标、故事核、叙事边界
├── 02_人物档案.md     # 只记录本篇实际人物
├── 03_历史资料.md     # 史料等级、地域与待考项
├── 04_小节大纲.md     # 小节蓝图与目标字数
├── 05_伏笔与双解.md   # 线索、理性解释与志异解释
└── 正文.md            # 当前正文
```
