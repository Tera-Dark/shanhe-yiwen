# 世界碎片仓

> 网页「**世界**」栏的**仓库唯一源**。  
> 这里只放**碎片**——设定短卡、杂记、随笔、书信摘、规矩/忌讳钉子——**不是**主线正文，也不是人物志全文。  
> 规范：`stories/catalog.schema.md` · 复杂度闸 `docs/世界/29`。

## 边界（必读）

| 放这里 | 不放这里 |
|---|---|
| 可点短卡（≤80 行说书气） | 主线 / 探游 / 传闻正文 → `stories/<地脉组>/` |
| 杂记、随笔、书信残页、市声摘录 | 人物完整档案 → `stories/人物志/`（N） |
| 卷内钉子（旗先过桥、假蹄、信息价…） | 时代总论、武学/组织长文 → `docs/` |
| catalog `world[]` 索引的条目 | 文学折 R → 地脉组内 |

**折写魂、志立档、碎片立卡**三套分立；网页世界栏以本目录 + `catalog.world` 为主，人物志仅作「人」分册入口。

## 目录

```text
stories/世界/
├── README.md          ← 本说明
├── 势力/              # kind: faction
├── 地标/              # kind: place（渡口、市口、碑、桥——非整卷地脉）
├── 生灵/              # kind: creature（传闻之兽、缺脸土地、假蹄…）
├── 物产/              # kind: material（旗杆、契纸、酱、刀鞘…）
├── 风物/              # kind: custom（规矩、忌讳、价码、已用话…）
└── 杂记/              # kind: scrap（随笔、书信残、市声札记、作者备忘可读化）
```

每条**一夹一卡**：`…/<显示名>/正文.md`（可前缀 `WLD-xx_`，忌空目录）。

开新卡：

```bash
python3 scripts/scaffold_entry.py --world --id WLD-XX --kind scrap --name 某杂记 --group hedong
```

## 正文头（轻 frontmatter）

```markdown
---
Version: 0.1
更新时间: 2026/7/21
id: WLD-QYBJ
kind: faction
name: 青驿镖局
fullness: 80
status: 成档
group: hedong
see:
  - G001
  - Z004
  - WLD-YUQ
  - docs/社会/07_组织势力.md
---

# 青驿镖局

一两段可读说明……
```

| 字段 | 说明 |
|---|---|
| `id` | `WLD-` 前缀 + 缩写，全局唯一 |
| `kind` | `faction` / `place` / `creature` / `material` / `custom` / `scrap` / `mystery` |
| `name` | 显示名 |
| `fullness` | 0–100，**成文充实度**（非战力、非进度条叙事） |
| `status` | 种子 / 框架 / 成档 |
| `group` | 可选地脉 `hedong` / `qiaotou` |
| `see` | **关联**：entry id · 其他 WLD- · docs 路径。网页点卡先开**本卡**，关联在侧栏/底栏 |

## 与 catalog

- `catalog.json` → `world[]`（schema 1.7+）为机器索引；改 path/id 须同步。  
- 未挂 catalog 时 integrity 可扫描本目录告警。  
- **禁止**只在 `web/app.js` 写死清单当唯一源。

## 网页怎么看

- 导航「世界」：按分册浏览碎片网格。  
- **点卡** → 碎片阅读器（非主线程目）：正文 + **关联**（另见正文 / 另见碎片 / 卷宗）。  
- 人物分册：已立志 N 优先；正文提及折叠为「名录」。  
- 路由：`#/world/:tab` · `#/shard/:WLD-id`

## 文风（魂类碎片）

卡面是**模糊介绍**，不是作者工单，也不是章节大纲。

- **像物品描述 / 耳语 / 说书残页**：气味、痕迹、人怎么说；两说并存，作者不盖章超常真相。  
- **禁止**写给作者看的句子：`W001 可写`、`正文只给…`、`沈陌可以…`、`别盖章…`、`主线探游共用时…`、`Z005 入城…`、`本卷不写…`。  
- 闸门（两本账、金庸式上限）写在 `docs/` 与 AGENTS，**不进卡面**。  
- `see` 挂章 / 其他碎片 / 卷宗；网页关联面板负责跳转。  
- 说书气短文，≤80 行；勿「账 / 武 / 禁 / 叙事」小标题。  
- 杂记夹可更散（市声、书信残），仍忌工程备注。
