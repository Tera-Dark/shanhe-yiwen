# 河东已用话

只收正文里已经出现、或马上要用稳的词。不从俗语总表整页搬【待考】黑话。新词先写进章，再回填这里。

**称人：** 走路的——脚夫懒称沈陌。沈半程——茶棚细者，走半程问半程。家书——护包时的嘴严口径。冷旗——夜喊、路上旗手的路名。茶白 / 白老头——棚里说书的，当面可不点破。

**规矩口：** 勿夜行。出事莫告官。旗要净 / 旗先过桥。冷的好。茶冷如树。入龛是债。一句一价。

**路与险：** 过我眼。脊是假硬。厚锣 / 薄锣。靴底有钉。三处勿入——康后院、北岔夜、顺手看册。

日常大白话优先。上表是落点，不是每句必念。十八卷里「亮青子、扯呼」一类未进河东正文前，慎用。神秘角色话可短，仍须一句听懂。
