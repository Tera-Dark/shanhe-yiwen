# 银与脚程

河东线上开口价，是脚夫店家嘴里的数，不是官簿。

饼一文上下，汤一两文，睡板三文，热水有时另加半文。半截碑过眼，一人两三文。茶棚一句正经话，另开，不跟续水混算。一小袋碎银，够数日住吃和过路，装不成阔。镖和酱车的账，走路的只闻不碰。

荷包瘪的日子比鼓的日子多。付不起就绕、就少问、就少住好板。欠什么往往比数字要紧：欠一拽，欠一句，欠一个眼色。

开口一个数，跟得上就行。
