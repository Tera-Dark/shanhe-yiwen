# 清洛卷 · 中篇总纲

> **现行唯一正文章卷。**  
> 声纹法源：主笔样板《入卷·清洛》（天不亮，雾先起……）。  
> 慢，且腻，且软——跟镇上的水一个脾气。不写网文升级。

## 一、一句话

清洛水软，人心不一定。顾晚棠出门送一封欠了三年的帖——帖背后欠的不只是钱——送完，还得回家。

## 二、女主

| 项 | 内容 |
|---|---|
| 名 | **顾晚棠**（棠姑娘 / 顾家的） |
| 路子 | 顾记镖行少东家：有庄、有人、有债、有好奇心 |
| 本事 | 记账、看路、几手防身；上不得台盘 |
| 想要 | 门楣上的诚信不旧；被人记得是顾晚棠 |
| 怕 | 庄垮、人散、夜里把名字应丢了 |
| 成长 | 懂 / 债 / 名 |

## 四、文笔声纹（强制）

1. **说书往下道**——段要顺，场面写满；允许卖关子，不碎成短句工单。  
2. **水软人未必软**——气味、门楣、闲话、灯；道理藏在动作里。  
3. **她不是侠**——柴米油盐里出门；女侠传说只当碎嘴。  
4. **异闻双解**——桥洞回声、夜呼名：风/人/醉鬼并存，作者不盖印。  
5. **余味**——可落在灯、花、水、半句市井闲话；不写哲理补丁。
