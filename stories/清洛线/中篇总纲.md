# 清洛卷 · 中篇总纲（试笔）

> **与河东并列，不替代沈陌主脊。**  
> 读者向：一位女少东家第一次把「家」背在身上出门。  
> 文案向：**燕云十六声**式地名与画面感 + **逆水寒**少东家的人情与选择；不写网文升级。

## 一、一句话

清洛水软，人情不软。顾晚棠出门不是为称霸，是为把一封过期的镖帖、一笔人情、一桩「说不清」的异闻，一并送过烟水桥——送完，还得回家。

## 二、女主

| 项 | 内容 |
|---|---|
| 名 | **顾晚棠**（路人称「棠姑娘」「顾家的」） |
| 路子 | 清洛顾记镖行少东家（逆水寒/燕云少东家感：有庄、有人、有债、有好奇心） |
| 本事 | 会记账、会看路、会几手防身；金庸上限内，不靠境界 |
| 想要 | 把庄撑住；被人记得是「顾晚棠」，不是「顾家那个丫头」 |
| 怕 | 庄垮、人散、自己变成只会躲在照壁后头的人 |
| 成长 | 懂 / 债 / 名（同 `docs/原则/25`，性别与声纹不同） |

## 三、地脉气味（燕云向）

- **水**：清洛河、烟水桥、梨花渡  
- **宅**：晚棠庄照壁、棠花、灶间葱油  
- **市**：灯市软声、茶摊半句话、镖旗不张扬  
- **异闻**：只留影——桥洞回声、过期帖、有人说「夜里别应声」；作者不盖章  

忌：河东「泥/碑/勿夜行」复读；忌章尾「泥还在转」式升华。

## 四、文笔声纹（本卷强制）

1. **先画面，后道理**——瓦、水、灯、衣角、气味；道理若有，藏在人物动作里。  
2. **短句可诗，段要顺**——允许一两句像联，但整章要能一口气读完，不碎成工单。  
3. **对话有温度**——市井、庄里人、江湖客各有口吻，遮住名字仍能认。  
4. **少作者旁白**——不解释「这叫江湖」；让读者看见。  
5. **余味，不说教**——章末落在一个具体动作或一句未说完的话。
