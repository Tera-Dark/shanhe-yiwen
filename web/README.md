# 山河异闻 · 地脉浏览台

本地只读浏览：按**地脉组**看江湖，按体例/气味/人物过滤，正文用**全屏沉浸阅读**（顶栏切换，无侧栏抽屉）。

## 启动

仓库根目录双击 `启动世界观网页.bat`，或：

```bash
npm start
```

→ `http://127.0.0.1:4182/`（需 Node.js ≥18）。

线上部署见 [DEPLOY.md](./DEPLOY.md)（Vercel 一键）。

## 导航

| 页 | 作用 |
|---|---|
| **入世** | 总览、地脉卡片、新近成稿 |
| **地脉** | 一级分类：河东线、桥头震后… |
| **体例** | 主纪/旁纪/残章/奇遇/探游/传闻/人物折/规矩帖/意象志 |
| **卷宗** | `docs/` 设定与规范 |
| **时序** | 嘉靖—崇祯年号轴、未解之谜 |

## 阅读

全屏阅读器：顶栏返回 / 入世 / 上一条 / 下一条 / 关闭；同组子导航；文末「另见」。快捷键 `Q` 检索，`Esc` 关闭，`←` `→` 翻篇。

## 数据

- `stories/catalog.json` — 地脉组 + 条目
- `/project/<相对路径.md>` — 正文与 `docs/` 设定
- `/__project-index` — 全库 Markdown 索引（排除 `archive/`）
- `/stories/*` — catalog 等静态资源

不写回仓库。改稿请直接改 Markdown 与 catalog。

## 视觉

纸本墨色 + 旧金点缀；Noto Serif/Sans SC；无游戏 HUD、无侧栏抽屉。
